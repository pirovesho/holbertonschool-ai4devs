### User Story 1
As a corporate employee, I want to find coworkers with similar commute routes so that I can share rides and reduce my commuting costs.

**Acceptance Criteria**:
- User enters home location and work location.  
- System suggests coworkers with similar routes and schedules.  
- User can send a carpool request to a suggested match.

**Priority**: MVP


### User Story 2
As a corporate employee, I want to create or join a carpool group so that I can regularly commute with the same colleagues.

**Acceptance Criteria**:
- User can create a carpool group with route and schedule details.  
- Other employees can request to join the group.  
- Group members can view shared ride schedules.

**Priority**: MVP


### User Story 3
As a corporate employee, I want to see available drivers or passengers near me so that I can quickly arrange a shared ride.

**Acceptance Criteria**:
- System shows nearby employees offering or requesting rides.  
- Distance and estimated pickup time are displayed.  
- User can send or accept ride requests.

**Priority**: High


### User Story 4
As a corporate employee, I want to rate my carpool partners so that reliable users are easier to identify.

**Acceptance Criteria**:
- After a completed ride, the system prompts users to rate each other.  
- Ratings are stored and shown on user profiles.  
- Users can view average ratings before joining a ride.

**Priority**: Medium


### User Story 5
As an HR manager, I want to track employee participation in carpooling so that I can evaluate sustainability initiatives.

**Acceptance Criteria**:
- Dashboard shows number of active carpool users.  
- Reports summarize trips and participation rates.  
- Data can be exported for sustainability reporting.

**Priority**: High


### User Story 6
As an HR manager, I want to see environmental impact statistics so that the company can measure reduced emissions.

**Acceptance Criteria**:
- System calculates CO₂ emissions saved from shared rides.  
- Dashboard displays total emissions reduction over time.  
- Reports can be generated monthly or yearly.

**Priority**: Medium


### User Story 7
As a corporate employee, I want notifications about ride updates so that I stay informed about schedule changes or requests.

**Acceptance Criteria**:
- User receives notifications when a ride request is accepted or declined.  
- System alerts users about schedule changes.  
- Notifications appear in the mobile app.

**Priority**: High


### User Story 8
As an HR manager, I want to see cost savings analytics so that the company can understand the financial benefits of carpooling.

**Acceptance Criteria**:
- System calculates estimated fuel savings from shared rides.  
- Dashboard shows total savings for employees and the organization.  
- Reports can be viewed over selected time periods.

**Priority**: Low
